# windows 命令

## 环境变量

显示

```shell
> echo $env:Path
```

显示

```shell
> $Env:<variable-name>
```

更改

```shell
> $Env:<variable-name> = "<new-value>"
```

::: info
[关于环境变量](https://learn.microsoft.com/zh-cn/powershell/module/microsoft.powershell.core/about/about_environment_variables?view=powershell-7.4)
:::
